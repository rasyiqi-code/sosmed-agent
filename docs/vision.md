# Product Vision: socmed-agent

## The Core Philosophy: "Set and Forget"
The socmed-agent is not a traditional SaaS dashboard or a scheduling tool like Buffer or Hootsuite. It is a true **Autonomous Digital Twin**. The primary goal is to provide **peace of mind** through minimal intervention. The user acts as the Architect, setting the rules, while the AI acts as the Executor.

## Design Aesthetic: "Architectural Minimalist"
- **Native Floating Window**: The UI abandons the standard web "Top Header + Sidebar" paradigm for a premium, single-window layout reminiscent of native macOS/iPad applications.
- **High-Density Data Context**: Utilizing tight typography (`8px` baselines with `inter` font) to display complex CRM and network states without feeling overwhelming.
- **Zero Friction Inputs**: Instead of complex workflow builders (like n8n), strategy adjustments are made through simple natural language inputs ("Global Steering").

## The 6 Core Pillars
1. **Command Center**: The primary dashboard displaying growth spectrums and the "Digital Twin Confidence" score.
2. **Campaign Engine**: The macro-controller for initializing multi-week thematic content streaks.
3. **The Studio**: The manual intervention chat, allowing the user to break the "forget" cycle and steer the AI.
4. **Chronology**: The exact timeline of what was published and what is queued.
5. **The Cortex**: The vector knowledge base where PDFs, URLs, and shower thoughts are dropped and seamlessly ingested by the AI.
6. **Network CRM**: The relationship tracker monitoring how the AI autonomously interacts with VIP accounts and leads.
